Top:
    Calculates the sales tax on the total collections for a month

First refinement:
    INITIALIZE variables
    CALCULATE sales tax
    DISPLAY results

Second refinement:
    SET float county_rate to 0.05
    SET float state_rate to 0.04

    DEFINE float total_collection, county_tax, state_tax, total_tax, and sales
    DEFINE character array month with 50 character-limit

    WHILE total_collection is not equal to -1 DO
        GET total_collection
        IF total_collection is equal to -1 THEN
            BREAK loop
        ENDIF
        GET month
        SET county_tax to total_collection * county_rate
        SET state_tax to total_collection * state_rate
        SET total_tax to county_tax + state_tax
        SET sales to total_collection - total_tax
    ENDWHILE

    PRINT total_collection
    PRINT sales
    PRINT county_tax
    PRINT state_tax
    PRINT total_tax
